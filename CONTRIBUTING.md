# Contributing

This is a personal opinionated Next.js + AWS deploy template. Feedback and corrections welcome via Issues; PRs are reviewed when I have time.

For larger changes, please open an Issue first to discuss the direction.

## Local development

See README.md for full setup. Quick checks before opening a PR:

- `npm install` runs cleanly
- `npm run lint` passes
- `npm run build` succeeds locally
- `docker compose up --build` produces a working image
- No new dependencies without justification in the PR description
